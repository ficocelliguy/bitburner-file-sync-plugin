# Support

Thanks for using the **Bitburner File Sync Plugin**. This page lists the supported ways to get help, report problems, or request features.


## Questions and discussion

For general questions about how to use the extension, please open a GitHub issue with the `question` label:

- https://github.com/ficocelliguy/bitburner-file-sync-plugin/issues

Alternatively, you can ask for help on the [official game Discord](https://discord.gg/TFc3hKD).


## Reporting bugs and requesting features

Please open an issue on GitHub:

- Issue tracker: https://github.com/ficocelliguy/bitburner-file-sync-plugin/issues
- New bug report: https://github.com/ficocelliguy/bitburner-file-sync-plugin/issues/new

When filing a bug, it helps a lot to include:

- The extension version (see the Extensions view in VS Code).
- Your VS Code version and operating system.
- The Bitburner version / source (Steam, web, etc.).
- A short description of what you expected to happen and what actually happened.
- Any relevant output from the **Bitburner Sync** output channel and from the VS Code Developer Tools console (`Help` → `Toggle Developer Tools`).

## Documentation

## Support scope

This extension is a community project maintained on a best-effort basis. There is no SLA, and responses may take some time. Pull requests are welcome.
